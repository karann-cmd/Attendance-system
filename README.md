# Face Recognition Attendance System

Real-time attendance system using face recognition via webcam.

## Tech Stack
- Python
- OpenCV
- DeepFace
- Django
- SQLite

## Features
- Real-time face detection via webcam
- Auto attendance marking in database
- Web dashboard with search & date filter
- Delete records feature
- Dark purple aesthetic UI

## Setup
pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
